¿Qué diferencia hay al insertar JavaScript en XHTML?

La diferencia principal al insertar JavaScript en XHTML radica en la necesidad de encerrar el código JavaScript entre etiquetas <![CDATA[...]> para evitar problemas con los caracteres especiales < y &, algo que no es necesario en HTML convencional.

¿Qué es el DOM y por qué fue diseñado?

Una interfaz de programación: El DOM es una interfaz estándar que define cómo los programas pueden interactuar con un documento web. Fue disenado con el objetivo de permitir a los desarrolladores acceder y manipular cada parte de un documento web con javascript.
